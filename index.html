Configurateur Plexiglass

Vue.js · WebGL · Canvas

Contexte du projet

Ce projet consistait à développer un outil de configuration en temps réel pour les plaques de plexiglass. L'objectif était de permettre aux utilisateurs de personnaliser complètement leurs plaques en sélectionnant différentes formes, en modifiant les dimensions, en ajoutant des trous et en effectuant des découpes intérieures, avec un aperçu 3D/2D instantané.

Solutions utilisées

  • Vue.js — Framework pour l'interface utilisateur réactive
  • Canvas & WebGL — Rendu 2D/3D pour la prévisualisation
  • Formes variables — Rectangle, Cercle, Étoile, Diamant et autres
  • Modificateurs — Dimensions, trous, découpes intérieures
  • Export & Configuration — Sauvegarde des configurations en temps réel

BiblioTech — Library Management

Laravel 12 · PHP 8.3 · SQLite

Contexte du projet

BiblioTech est une application complète de gestion de bibliothèque en ligne. Ce projet aéronautique démontre une expertise full-stack en Laravel 12 avec une architecture MVC professionnelle. L'application propose un système complet d'emprunt de livres, de réservations, ainsi que des fonctionnalités avancées comme la gamification et le système de notation.

Fonctionnalités principales

  • Gestion de catalogue — Système complet de gestion de livres avec catégories et recherche
  • Système d'emprunt — Emprunts, retours, historique et gestion des délais
  • Réservations — Système de réservation pour les livres non disponibles
  • Authentification — Authentication sécurisée avec rôles (Admin, Librarian, User)
  • Gamification — Système de points, badges et classement des utilisateurs
  • Avis et notes — Système de notation et d'avis utilisateurs
  • Tableau de bord admin — Interface pour gérer la bibliothèque avec statistiques en temps réel
  • Notifications email — Alertes automatiques pour les retours et réservations

Stack technologique

  • Laravel 12 — Framework PHP moderne avec architecture MVC
  • PHP 8.3 — Programmation orientée objet, namespaces et features modernes
  • SQLite 3 — Base de données relationnel portable
  • Eloquent ORM — Gestion élégante des modèles et relations
  • Blade Templates — Moteur de templating puissant
  • Bootstrap 5 — Framework CSS responsive et moderne
  • Spatie Permission — Gestion des rôles et permissions
  • PHPUnit — Tests automatisés (80%+ couverture)
  • GitHub Actions — CI/CD pipeline automatisée
  • Docker — Containerisation pour production

Compétences démontrées

  • Architecture logicielle — Patterns MVC, SOLID, DRY
  • Base de données — Migrations, relations Eloquent, optimisations
  • Backend — Contrôleurs resource, validation, middleware
  • Frontend — Blade templates, Bootstrap, CSS responsive
  • Sécurité — CSRF, XSS, SQL injection protection, authentification
  • Testing — Tests unitaires et d'intégration, coverage tracking

BoutikPro — Commerce Management

Python · MySQL · SQLAlchemy · UML

Contexte du projet

BoutikPro est une application de gestion commerciale développée comme évaluation en contrôle continu (CCF) pour le BTS SIO SLAM. Le projet adresse un besoin professionnel réel : une PME e-commerce souhaitant remplacer son système manuel fragmenté par une application Python intégrée. L'application gère clients, commandes, produits, programmes de fidélité et relations fournisseurs, démontrant une maîtrise complète de la modélisation de données et du développement backend.

Objectifs & Démarche

  • Analyse métier — Traduction des besoins professionnels en diagrammes UML de cas d'usage
  • Modélisation données — Conception des modèles conceptuels (MCD) et relationnels (MLD)
  • Design base données — Implémentation du schéma MySQL avec 11 tables et gestion des contraintes
  • Développement Backend — Création d'application Python avec opérations CRUD complètes
  • Approches multiples — Démonstration de 3 méthodes d'accès BD (DB-API, SQLAlchemy Core, ORM)
  • Requêtes avancées — Conception de jointures complexes et fonctions d'agrégation pour business intelligence

Fonctionnalités principales

  • Gestion clients — Affichage, création, modification, suppression de clients
  • Gestion produits — Catalogue organisé par catégories
  • Gestion commandes — Enregistrement des commandes avec lignes multiples
  • Facturation — Génération et affichage des factures
  • Programme de fidélité — Suivi des points et cartes clients
  • Gestion fournisseurs — Administration des fournisseurs et leurs catégories
  • Recommandations — Suivi des relations de recommandation entre clients
  • Requêtes avancées — Requêtes complexes avec jointures et agrégations

Stack technologique

  • Python 3 — Langage orienté objet avec gestion d'erreurs robuste
  • MySQL 8+ — Base de données relationnelle avec 11 tables
  • SQLAlchemy — ORM moderne et abstraction de requêtes SQL
  • PlantUML — Modélisation UML des cas d'usage et diagrammes
  • Docker — Containerisation pour Codespaces et environnements locaux
  • MySQL Connectors — mysql-connector-python et PyMySQL
  • Git/GitHub — Versioning et collaboration

Compétences démontrées

  • Modélisation UML — Diagrammes de cas d'usage, identification des acteurs
  • Conception base données — MCD, MLD, normalisation, schéma relationnel
  • SQL avancé — Jointures, aggrégations, contraintes, transactions
  • Programmation Python — CRUD, connectivité BD, gestion d'erreurs
  • Patterns d'accès données — Connaissance de 3 approches distinctes
  • Analyse métier — Traduction des besoins professionnels en spécifications techniques
  • Infrastructure — Configuration Docker, Codespaces, multi-plateforme support
