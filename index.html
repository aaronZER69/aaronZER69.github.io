Site J-Pop — YOASOBI

HTML · CSS · JavaScript

Contexte du projet

Ce projet avait pour objectif de créer un site vitrine dynamique et attrayant dédié au groupe musical japonais YOASOBI. Le site présente les membres du groupe, leur histoire, discographie et offre une expérience utilisateur immersive avec des animations et des interactions fluides.

Solutions utilisées

  • HTML5 — Structure sémantique et moderne du site
  • CSS3 — Design responsive avec animations et effets visuels
  • JavaScript — Interactivité, navigation fluide et gestion des événements
  • Responsive Design — Compatible sur tous les appareils (mobile, tablette, desktop)

Gestion de parc informatique — GLPI

GLPI · MySQL · Réseau

Contexte du projet

Ce projet consistait à administrer et gérer un parc informatique d'entreprise via GLPI (Gestion Libre de Parc Informatique). L'objectif était de mettre en place une solution complète d'inventaire, de gestion des tickets d'assistance, et du suivi des équipements informatiques pour assurer une meilleure organisation et traçabilité.

Solutions utilisées

  • GLPI — Plateforme open-source de gestion de parc informatique
  • MySQL — Base de données pour la persistence des données
  • Réseau — Configuration et gestion du réseau informatique
  • Inventaire — Identification et suivi des équipements (PC, imprimantes, serveurs)
  • Ticketing — Système de tickets pour les demandes d'assistance

Site E-Commerce — Huiles

React · Tailwind · Node.js

Contexte du projet

Ce projet consistait à créer une plateforme e-commerce complète dédiée à la vente d'huiles de qualité. L'objectif était de proposer une expérience d'achat fluide et professionnelle avec un catalogue riche, un système de panier, et une gestion de commandes pour offrir aux clients une solution d'achat en ligne moderne et sécurisée.

Solutions utilisées

  • React — Framework front-end pour une interface dynamique et réactive
  • Tailwind CSS — Styling moderne et responsive
  • Node.js — Backend pour la gestion des données et des commandes
  • Panier d'achat — Gestion du panier avec persistance en local storage
  • Système de paiement — Intégration pour les transactions sécurisées
